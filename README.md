# react-bill-program 记账单功能

## 路径解析配置

1.安装craco

`npm i -D @craco/craco`

2.项目根目录下创建配置文件

`craco.config.js`

3.配置文件中添加路径解析配置

4.包文件中配置启动和打包命令

## vscode的联想路径配置（vscode会自动读取配置帮助我们自动联想提示）

1.根目录下新增配置文件

`jsconfig.json`

2.添加路径提示配置

## 数据Mock
